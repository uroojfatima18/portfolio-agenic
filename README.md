# Portfolio Agenic - Premium AI Agent Portfolio

This is a premium-grade portfolio project built with **Vite**, **Vanilla JS**, and **Stitch MCP** for high-fidelity UI designs.

## 🚀 Getting Started

Follow these commands to run the project locally on your machine:

### 1. Install Dependencies
Run this command first to install all necessary packages (Vite, etc.):
```bash
npm install
```

### 2. Run Locally (Development)
To start the local development server and see your portfolio in the browser:
```bash
npm run dev
```
After running this, copy the URL (usually `http://localhost:5173`) and paste it into your browser.

### 3. Production Build
To create a minimized, high-performance version of your site for deployment:
```bash
npm run build
```

## 🤖 AI Skills & Stitch MCP

This project is integrated with **Stitch Design Expertise**. 

### How to use the `.agents` Skill:
The `.agents/stitch-design` folder contains the "Brain" for UI/UX improvements. To use it, simply ask the AI Assistant:
*   *"Review my design using the stitch-design skill."*
*   *"Enhance my portfolio structure based on the skill documentation."*

### How to use Stitch MCP:
Stitch is used to generate premium UI sections. You can ask for:
*   **Design Generation:** *"Generate a premium dark-mode hero section for an AI developer."*
*   **Design Edits:** *"Edit the current project cards to include glassmorphism and glow effects."*

## 📂 Project Structure
- `/src`: Contains the main logic (`main.js`) and premium styles (`style.css`).
- `.agents/`: Contains the specialized instructions for AI design workflows.
- `index.html`: The main entry point of your portfolio.
