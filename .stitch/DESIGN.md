# Agentic Portfolio Design System

## Atmosphere & Vibe
- **Conceptual Theme**: "Artificial Intelligence Command Center"
- **Style**: Futuristic, High-Contrast Dark Mode with Glassmorphism effects.
- **Engagement**: Low-latency animations, glow effects on interactive elements, and data-driven visuals.

## Design Tokens

### Color Palette
- **Deep Space Black** (#050505): Used for main page background.
- **Surface Layer** (rgba(255, 255, 255, 0.03)): Used for glassmorphic cards and sections.
- **Electric Cyan** (#22D3EE): Primary branding color for buttons, active states, and highlights.
- **Plasma Purple** (#8B5CF6): Secondary color for gradients and "Agentic Thinking" indicators.
- **Cyber Mint** (#10B981): Used for success states and "System Live" indicators.

### Typography
- **Primary Header**: 'Outfit', sans-serif (Bold, -2% letter spacing).
- **Secondary / Technical**: 'JetBrains Mono', monospace (Used for code snippets, tags, and data feeds).
- **Base Text**: 'Inter', sans-serif (Regular/Medium).

### Geometric Language
- **Corner Roundness**: 12px (Soft-UI standard).
- **Borders**: 1px solid rgba(255, 255, 255, 0.1) with 30px backdrop-blur.
- **Shadows**: Subtle glow (box-shadow: 0 0 20px rgba(34, 211, 238, 0.15)).

## Component Guidelines
- **Navigation**: Sticky pill-shaped bar with background blur.
- **Cards**: Minimalist with subtle scale-up on hover and glowing borders.
- **Buttons**: Full cyan fills for primary CTAs; ghost buttons with cyan borders for secondary.
